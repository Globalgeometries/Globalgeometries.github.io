Project Title: Global Geometries
Medium(s): Creative Coding, Interactive Fiction, Time-Based Media, Archive
One Line Description: Global Geometries: A Digital Library of Interactive, Audio-Reactive Pattern Art in p5.js

Description: “Global Geometries” is an open-source, digital library of interactive, culturally-inspired geometric patterns built using p5.js. It combines generative art, mathematics, and sound interaction to animate traditional motifs. From Islamic tessellations to Indian mandalas, sketches respond to real-time audio input.
Each piece in the library will include lessons, tutorials, and context linking mathematical concepts to culture. This platform will support remixing, open education, and creative collaboration.

Real-time Data Use: The project maps rhythm, pitch, and amplitude from audio to dynamic geometric transformations—turning cultural motifs into responsive, immersive experiences.

Software Stack: Built entirely using p5.js and p5.sound for live interactivity. Patterns will be programmed for modular reuse and mapped to audio via the sound library for responsive feedback.

Fellowship Questions: How can we make audio-reactive generative art accessible for learners? What are the best ways to integrate culture, math, and code in creative learning? The Processing Foundation’s support will help us design inclusive pedagogy and scale community engagement.
